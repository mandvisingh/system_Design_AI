# Advanced System Design & AI Curriculum (Python + JavaScript)
### Format: 1 YouTube Video • 1 Assignment • 1 Paper/Blog

---

## Weeks 1–2: Vision & Fundamentals

### Week 1
- **YouTube:** [Gaurav Sen – Intro to System Design](https://www.youtube.com/watch?v=r1MXT5j8vGk)
- **Assignment:** Write your learning manifesto, listing 3 world-leading systems (AI/scalable)—analyze their core architecture.
- **Paper/Blog:** [System Design Primer](https://github.com/donnemartin/system-design-primer)

### Week 2
- **YouTube:** [StatQuest – Intro to AI & ML](https://www.youtube.com/watch?v=aircAruvnKk)
- **Assignment:** Map out monolithic vs microservices architecture; record a 10-min teach-back screencast.
- **Paper/Blog:** [Polyglot Persistence](https://martinfowler.com/bliki/PolyglotPersistence.html)

## Weeks 3–10: Deep System Design

### Week 3
- **YouTube:** [Google – Distributed Systems](https://www.youtube.com/watch?v=YwG_6u6wZg4)
- **Assignment:** Build/diagram a distributed log service (Python async); review CAP theorem.
- **Paper/Blog:** [Designing Data-Intensive Applications (Book Excerpt)](https://dataintensive.net/)

### Week 4
- **YouTube:** [Corey Schafer – PostgreSQL with Python](https://www.youtube.com/watch?v=HXV3zeQKqGY)
- **Assignment:** Design & implement multi-model data platform (Postgres/Mongo/Redis).
- **Paper/Blog:** [MongoDB Data Patterns](https://www.mongodb.com/blog/post/building-with-patterns-a-summary)

### Week 5
- **YouTube:** [Trevor – REST vs GraphQL vs gRPC](https://www.youtube.com/watch?v=zpUuR2_G0lA)
- **Assignment:** Benchmark high-throughput API (Python); JS stress-test client and report results.
- **Paper/Blog:** [Building Backend APIs at Stripe](https://stripe.com/blog/backend-engineering)

### Week 6
- **YouTube:** [Gremlin – Chaos Engineering Explained](https://www.youtube.com/watch?v=1vnF3qatvFQ)
- **Assignment:** Implement circuit breakers, failover; run a chaos experiment and write findings.
- **Paper/Blog:** [Principles of Chaos Engineering](https://principlesofchaos.org/)

### Week 7
- **YouTube:** [Fireship – JWT Authentication Explained](https://www.youtube.com/watch?v=7Q17ubqLfaM)
- **Assignment:** Build secure auth for multi-tenant app (Python + JS with JWT); demo an exploit+fix.
- **Paper/Blog:** [OWASP Top 10 (2025)](https://owasp.org/www-project-top-ten/)

### Week 8
- **YouTube:** [Cloudflare – What is a CDN?](https://www.youtube.com/watch?v=HkYxmaWYFYk)
- **Assignment:** Architect a global ML inference API; JS: visualize edge latency.
- **Paper/Blog:** [AWS Lambda@Edge Whitepaper](https://docs.aws.amazon.com/lambda/latest/dg/lambda-edge.html)

### Week 9
- **YouTube:** [AWS – Common Failure Modes](https://www.youtube.com/watch?v=3mZaNBx6lvc)
- **Assignment:** 15-min deep-dive presentation: Scaling failures and lessons learned.
- **Paper/Blog:** [Uber Eng – Scaling Uber](https://eng.uber.com/scaling-uber/)

### Week 10
- **YouTube:** [Ben Awad – Full-stack App from Scratch](https://www.youtube.com/watch?v=I6ypD7qv3Z8)
- **Assignment:** Build a fault-tolerant scalable API (Python backend, React frontend, Redis, JWT).
- **Paper/Blog:** [Martin Fowler – Microservices Guide](https://martinfowler.com/articles/microservices.html)
